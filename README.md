# Backtest Engine

A modular Python backtesting engine for systematic trading strategies.

This project focuses on:
- Clean system architecture
- Realistic execution modelling
- Transaction cost and slippage modelling
- Risk metrics and performance analytics

## Features

- Strategy abstraction layer
- Portfolio management logic
- Transaction cost modelling
- Performance reporting (Sharpe, drawdown, volatility)
- Extensible architecture for new strategies

## Example Workflow

Data → Signal Generation → Position Sizing → Execution Simulation → Performance Analysis

## Tech Stack

- Python 3.11+
- Pandas / NumPy
- Matplotlib
- Pytest (unit testing)
